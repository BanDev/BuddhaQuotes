## Features
- AMOLED mode
- Over 340 quotes!
- Favourite quotes
- Free, libre and open source
- No ads
- No trackers (verified using [ClassyShark3xodus](https://bitbucket.org/oF2pks/fdroid-classyshark3xodus/src/master/))
- No non-free dependencies like Google services
- Completely offline
- Small file size!

## Contributing
